# Commands Used - Day 10

## Create Scripts

```bash
touch functions-demo.sh
touch calculator-args.sh
touch menu.sh
touch health-check.sh
```

## Edit Scripts

```bash
nano filename.sh
```

## Make Executable

```bash
chmod +x filename.sh
```

## Execute Scripts

```bash
./filename.sh
```

## Command-Line Arguments

```bash
./calculator-args.sh 20 30
```

## Linux Commands Used

```bash
whoami
hostname
date
pwd
df -h
free -h
uptime
```
