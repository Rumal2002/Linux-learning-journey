# Day 10 - Advanced Bash Scripting

## Overview

On Day 10 of my DevOps learning journey, I completed the Bash Scripting Fundamentals phase by learning advanced Bash concepts and applying them through hands-on automation projects.

This lab focused on writing reusable Bash scripts using functions, accepting command-line arguments, creating interactive menus with case statements, and building a Linux System Health Checker that collects essential system information.

These concepts are fundamental for Linux administration, automation, DevOps engineering, and CI/CD workflows.

## Mini Project

### Linux System Health Checker

This project generates a basic Linux system report by displaying:

- Current User
- Hostname
- Current Date & Time
- Disk Usage
- Memory Usage
- System Uptime

This project demonstrates how Bash can automate common Linux administration tasks used by DevOps and System Engineers.
