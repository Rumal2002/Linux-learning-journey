# Day 8 - Bash Scripting Fundamentals

## Overview

Today I started learning Bash Scripting, one of the most essential skills for DevOps Engineers.

Bash scripting allows Linux users to automate repetitive tasks, execute multiple commands through scripts, and simplify system administration.

Throughout this lab, I learned how to create executable Bash scripts, work with variables, receive user input, perform arithmetic operations, use conditional statements, create loops, and automate basic file management tasks.

---

## Topics Covered

- Introduction to Bash
- Shell vs Terminal
- Creating Bash Scripts
- Shebang (`#!/bin/bash`)
- File Execution Permissions
- Variables
- User Input
- Arithmetic Operations
- Conditional Statements (`if`)
- Loops (`for`)
- File Backup Automation
