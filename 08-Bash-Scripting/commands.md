# Commands Used - Day 8

## Create Folder

```bash
mkdir 08-Bash-Scripting
```

## Navigate

```bash
cd 08-Bash-Scripting
```

## Create Script

```bash
touch hello.sh
```

## Edit Script

```bash
nano hello.sh
```

## Display File

```bash
cat hello.sh
```

## Execute Script

```bash
bash hello.sh
```

## Make Executable

```bash
chmod +x hello.sh
```

## Execute Directly

```bash
./hello.sh
```

## Display Current Directory

```bash
pwd
```

## Current User

```bash
whoami
```

## Display Date

```bash
date
```

## Display Hostname

```bash
hostname
```

## List Files

```bash
ls
```

## Long Listing

```bash
ls -l
```

## Copy Files

```bash
cp
```

## Create Directory

```bash
mkdir
```
