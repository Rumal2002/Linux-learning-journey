#  Linux Server Administration Lab

##  Project Objective

This project simulates common Linux administration tasks performed by a Junior DevOps Engineer.

The goal was to apply Linux fundamentals in a realistic server environment instead of practicing commands in isolation.

---

#  Skills Demonstrated

- Linux File Management
- Directory Navigation
- File Permissions
- User & Group Administration
- Process Management
- System Monitoring
- Linux Command Line

#  Practical Tasks Completed

## File Management

- Created project directories
- Created configuration files
- Copied files
- Renamed files
- Removed unnecessary files

---

## File Permissions

Applied secure permissions using:

- chmod 755
- chmod 644
- chmod 600

---

## User Management

Created:

- DevOps Group
- Developer User

Assigned users to groups using Linux administration commands.

---

## Process Management

Practiced:

- Running foreground processes
- Background jobs
- Job management
- Process termination

---

## System Monitoring

Collected system information using:

- CPU
- Memory
- Disk Usage
- Logged-in User
- Hostname
