# Commands Used - Day 9

## Create Files

```bash
touch my-profile.sh
touch environment.sh
touch check-command.sh
```

## Edit Files

```bash
nano my-profile.sh
nano environment.sh
nano check-command.sh
```

## Display File Contents

```bash
cat filename.sh
```

## Make Scripts Executable

```bash
chmod +x filename.sh
```

## Execute Scripts

```bash
./filename.sh
```
