# Day 11 - Networking Fundamentals

## Overview

Today I began the Networking Fundamentals phase of my DevOps learning journey.

Networking is one of the most important foundations for DevOps, Cloud Computing, System Administration, and Cybersecurity. Understanding how devices communicate, how data travels across networks, and how Linux interacts with network services is essential before moving on to Docker, Kubernetes, AWS, and CI/CD.

During this lab, I explored networking concepts and practiced Linux networking commands to inspect system and network information.

## Practical Commands

The following Linux networking commands were practiced:

| Command | Description |
|---------|-------------|
| `hostname` | Display the system hostname |
| `whoami` | Display the current logged-in user |
| `hostname -I` | Display the system IP address |
| `ip addr` | Show network interfaces and IP configuration |
| `ip route` | Display the routing table |
| `ping google.com` | Test internet connectivity |
| `ping localhost` | Test local network communication |
| `nslookup google.com` | Perform DNS lookup |
| `ip -s link` | Display network interface statistics |
