#  What I Learned

## What is Linux?

Linux is a free and open-source operating system used in servers, cloud platforms, enterprise environments, and embedded systems. It is known for its stability, security, and flexibility, making it the preferred operating system for DevOps engineers.

---

## Why Linux for DevOps?

Most modern cloud infrastructure and production servers run Linux.

As a DevOps Engineer, Linux knowledge is essential for:

* Server administration
* Application deployment
* Automation
* Containerization
* CI/CD pipelines
* Infrastructure management

---

## Open Source

Linux is open-source software, meaning its source code is publicly available. Developers around the world can study, modify, improve, and distribute it.

---

## Linux Kernel

The Linux kernel is the core component of the operating system. It manages communication between software applications and computer hardware such as the CPU, memory, storage, and input/output devices.

---

## Linux Distribution

A Linux distribution combines the Linux kernel with system utilities, package managers, and additional software to create a complete operating system.

Examples include:

* Ubuntu
* Debian
* Fedora
* Rocky Linux
* AlmaLinux

---

## Shell

The shell is a command-line interpreter that receives user commands and communicates with the Linux kernel to execute them.

---

## Terminal

The terminal is the application used to interact with the shell through text-based commands.

---

## GUI vs CLI

### GUI (Graphical User Interface)

Uses windows, icons, and menus for interaction.

### CLI (Command Line Interface)

Uses text commands entered through the terminal.

CLI is preferred in DevOps because it is faster, lightweight, scriptable, and ideal for remote server management.

---

#  Commands Practiced

```bash
pwd
```

Displays the current working directory.

---

```bash
whoami
```

Displays the currently logged-in user.

---

```bash
hostname
```

Displays the system hostname.

---

```bash
date
```

Displays the current system date and time.

---

```bash
cal
```

Displays the current month's calendar.

---

```bash
uname -a
```

Displays detailed Linux system information.

* The shell interprets user commands.
