# Day 12 - IP Addressing Fundamentals

## Overview

Today I continued my Networking Fundamentals learning journey by studying IP addressing, one of the most important concepts in computer networking and DevOps.

I learned how devices are uniquely identified on a network, the differences between IPv4 and IPv6, how public and private IP addresses are used, and the purpose of subnet masks. I also explored Linux networking commands to inspect IP configurations and routing information.

These concepts form the foundation for understanding cloud networking, Docker, Kubernetes, AWS, and modern DevOps infrastructure.

---

## Topics Covered

- What is an IP Address?
- Why IP Addresses are Important
- IPv4
- IPv6
- Public vs Private IP Addresses
- Introduction to Subnet Masks
- Linux IP Configuration Commands

---

## Practical Commands

| Command | Description |
|---------|-------------|
| `hostname -I` | Display the system IP address |
| `ip addr` | Show detailed network interface configuration |
| `ip route` | Display the routing table |
| `hostname` | Display the system hostname |
| `cat /etc/resolv.conf` | Display configured DNS servers |
| `ip link` | Display available network interfaces |
