# Commands Used - Day 12

## Display IP Address

```bash
hostname -I
```

## Show Detailed Network Configuration

```bash
ip addr
```

## Display Routing Table

```bash
ip route
```

## Display Hostname

```bash
hostname
```

## Display DNS Configuration

```bash
cat /etc/resolv.conf
```

## Display Network Interfaces

```bash
ip link
```

## Display Current User

```bash
whoami
```
